Single-Neuron Network with and without PCA
Comparación del rendimiento de una red neuronal simple entrenada con datos originales versus datos reducidos mediante PCA (Principal Component Analysis).

Contenido
tarea_4_con_PCA_comparacion.ipynb - Red neuronal con reducción dimensional PCA
tarea_4_con_PCA_comparacion.html - Análisis comparativo interactivo
Objetivo
Evaluar cómo afecta la reducción de dimensionalidad mediante PCA al rendimiento y tiempo de entrenamiento de redes neuronales.

Resultados
Los notebooks generan visualizaciones comparativas que muestran:

Accuracy en datos originales vs. reducidos
Curvas de entrenamiento
Matriz de confusión
Tiempo de ejecución
Notas
Los archivos .keras contienen modelos entrenados
Los análisis se ejecutan en Keras/TensorFlow
